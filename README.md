# Innomatics Hackathon – Food Delivery Analysis

This project analyzes food delivery data to extract business insights using
Python, SQL, and data analysis techniques.

## Files
- FD_final.ipynb – Complete data analysis notebook
- final_delivery_dataset.csv – Dataset used for analysis
- sql.txt – SQL queries used
- README.md – Project documentation

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL
- Jupyter Notebook
